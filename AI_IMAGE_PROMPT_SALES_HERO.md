# AI Image Generator Prompt for Sales Page Hero Background

## Primary Prompt (for DALL-E, Midjourney, Stable Diffusion):

```
A sophisticated, elegant real estate background image for a property sales website hero section. The scene features a luxurious modern London property exterior with architectural details - elegant Georgian or contemporary architecture with clean lines. The color palette is dominated by deep navy blue tones (#1C2A39), charcoal grays, and subtle warm gold accents (#C0A060). The image should have a professional, high-end aesthetic with soft natural lighting - either early morning golden hour or late afternoon soft light. Include subtle architectural elements like elegant doorways, window frames, or property facades in the background, slightly out of focus to create depth. The overall mood should be premium, trustworthy, and aspirational. The image should be suitable for text overlay with dark gradient overlays. Soft bokeh effects, gentle shadows, and a slightly desaturated look. Aspect ratio 16:9, high resolution, professional photography style, cinematic composition.
```

## Alternative Shorter Prompt:

```
Luxury real estate property background, deep navy blue (#1C2A39) and charcoal tones with subtle gold accents (#C0A060), elegant London architecture, soft natural lighting, professional photography, cinematic, suitable for text overlay, slightly desaturated, premium aesthetic, 16:9 aspect ratio
```

## Detailed Style Specifications:

**Color Palette:**
- Primary: Deep navy blue (#1C2A39) - dark, sophisticated
- Secondary: Charcoal gray (#2F3A4A) - professional
- Accent: Warm gold (#C0A060) - luxury, premium
- Background: Dark tones that work with black overlays

**Style Keywords:**
- Professional real estate photography
- Luxury property aesthetic
- Sophisticated and elegant
- Clean, modern architecture
- Soft, natural lighting
- Slightly desaturated
- Cinematic composition
- Premium quality
- Suitable for text overlay

**Technical Requirements:**
- Aspect ratio: 16:9 (or 1920x1080)
- High resolution (at least 1920x1080px)
- Soft focus on background elements
- Dark enough to work with gradient overlays
- Not too busy - text needs to be readable

## For Midjourney specifically, add these parameters:

```
--ar 16:9 --style raw --quality 2 --v 6
```

## For Stable Diffusion, use these settings:

- Model: Realistic Vision or DreamShaper
- Steps: 30-50
- CFG Scale: 7-9
- Negative prompt: "bright colors, oversaturated, cluttered, busy, low quality, blurry"

## Notes:
- The image will have dark gradient overlays applied in CSS, so it should be dark enough to work with those
- Text will be white/light colored, so ensure good contrast
- Keep the image subtle - it's a background, not the main focus
- The gold accents should be subtle and elegant, not overwhelming
